Introducing Dalcloud

Dalcloud is an open cloud application engine that hosts and runs applications in any cloud infrastructures.

Platform

Dalcloud is a platform-as-a-service platform. You can easily write and deploy your applications with ones written and deployed by others with almost zero configuration.

No Lock-In

With dalcloud, you don't have to be locked in one service provider anymore. You can deploy dalcloud to any popular IaaSs, your private clouds, or even your own PCs.

Dependency Discovery

If your dalcloud application is depending on some other applications or services, it is automatically resolved when tested and/or deployed.

Controlled

In cloud settings, you can manage and scale your dalcloud nodes with simple scripting independent to the underlying IaaS provider. You can even use management REST APIs remotely if necessary.

Open Source

Dalcloud is an open source project. You can fork the project and use it however you want. Don't forget to contribute back to the community when possible though :)
